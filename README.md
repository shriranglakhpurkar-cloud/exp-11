🧪 Aim

To create a dataset and perform basic data analysis operations using the Pandas library in Python, including loading, exploring, and summarizing the dataset.

📖 Theory

Pandas is a powerful Python library used for data manipulation and analysis. It provides data structures like DataFrame and Series to handle structured data efficiently.

In this experiment, the following concepts are used:

Dataset Creation:

A dataset is manually created using a Python dictionary and converted into a DataFrame.

Data Loading:

Data is loaded from a CSV file using read_csv().

Data Exploration Functions:

shape → returns number of rows and columns

size → total number of elements

info() → summary of dataset

describe() → statistical summary

Data Inspection:

head() → first few rows

tail() → last few rows

sample() → random rows

Data Cleaning Checks:

isnull() → checks missing values

duplicated() → checks duplicate records

nunique() → counts unique values


These operations help in understanding the structure and quality of the dataset.
